📦 Amazon Web Scraping Project
📌 Project Overview

This project demonstrates how to perform web scraping on Amazon using Python.
It extracts product details such as:

🏷 Product Title

💰 Price

⭐ Ratings

📝 Reviews

🔗 Product URL

The collected data can be used for:

Price comparison

Market research

Data analysis

E-commerce insights

🛠 Technologies Used

🐍 Python

📦 requests

🍲 BeautifulSoup

📊 pandas

📓 Jupyter Notebook
